# Student Performance Tracker (EduTrack Pro)

An advanced web application prototype designed to monitor, track, and visualize student academic performance metrics.

## 🚀 Features Included
- **Simulated Authentication System:** Clean login page with hardcoded validation and failure state messages.
- **Responsive Navigation Bar:** Adaptive structural layout supporting mobile and desktop resolutions.
- **Dynamic Dashboard:** Displays real-time calculations of total registered users and statistical breakdowns.
- **Data Visualisation:** Includes a Chart.js implementation demonstrating subject core grade metrics.
- **Data View Inclusions:** Two discrete structural layouts showcasing datasets (Student List & Detailed Subject Grade Sheet).
- **CRUD Interaction:** Ability to add new students dynamically and persist them using browser LocalStorage.
